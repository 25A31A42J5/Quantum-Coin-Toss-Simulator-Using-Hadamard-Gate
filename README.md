# Quantum Coin Toss: A Qiskit-Powered Randomizer.
This repository contains a Quantum Coin Toss Simulator built with Python and Qiskit. While a standard computer uses math-based "pseudo-random" algorithms to flip a coin, this project taps into the actual behavior of subatomic particles to get a result. By using a Hadamard Gate, we shift a qubit into a state where it isn't just a 0 or a 1, but a perfect 50/50 mix of both simultaneously.

# How the Logic Works:
The Coil-Flip happens through a concept called superposition. In a classical system, a coin flip is technically predictable if you know the exact force and wind speed. In this quantum version, we start with a qubit in the |0> state and apply a Hadamard (H) Gate. This creates a mathematical state represented by a formula.When we finally measure the qubit, the superposition collapses into a definitive 0 (Heads) or 1 (Tails). Because this collapse is fundamentally random, the simulator provides a level of entropy that classical computers can't replicate on their own.
